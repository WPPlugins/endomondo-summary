== Description ==

Allows you to embed the Endomodo Summary widget anywhere on your wordpress site using a shortcode.

This is an active beta project, please report any bugs so I can fix them.

== Installation ==

1. Extract and Upload `endomondo-summary` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Fill in the variables on the Endomondo-Summary page, located in the Settings section on the admin menu
4. Place `[endomondo-summary]` in your post/page
